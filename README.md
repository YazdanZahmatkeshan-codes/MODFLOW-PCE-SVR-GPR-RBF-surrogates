# MODFLOW-PCE-SVR-GPR-RBF-surrogates
MODFLOW-2005 groundwater flow simulations coupled with PCE, SVR, GPR, and RBF surrogate models for uncertainty quantification and assessing output precision.


## Contents

### MODFLOW_heterogeneous model
Heterogeneous groundwater model using FloPy and MODFLOW-2005.

### Surrogate_Models
Polynomial Chaos Expansion (PCE), Gaussian Process Regression (GPR), Support Vector Regression (SVR), and Radial Basis Functions (RBF) surrogate models.

### Example_Data
Example datasets for reproducing the workflow.

## Requirements

Python 3.13

Packages:

```bash
pip install -r requirements.txt
```
